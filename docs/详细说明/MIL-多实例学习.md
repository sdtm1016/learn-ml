# MIL（多实例学习）

## 简介
多实例学习（Multiple Instance Learning，MIL）是一种弱监督学习范式，训练数据以"包"（bag）的形式组织，每个包包含多个"实例"（instance），只有包级别的标签。

## 核心概念
- **包（Bag）**：一组实例的集合
- **实例（Instance）**：包中的单个样本
- **标签分配**：只有包级别标签，实例标签未知
- **聚合函数**：将实例特征聚合为包特征

## 主要方法
- **基于实例的方法**：识别关键实例
- **基于包的方法**：直接建模包特征
- **注意力机制**：加权聚合实例特征

## 医疗应用场景

### 病理图像分析
- 全切片图像（WSI）分类
- 宫颈癌筛查
- 乳腺癌分级
- 前列腺癌诊断

### 医学影像检测
- 弱监督病灶检测
- 多病灶识别
- 异常区域定位

### 基因组学
- 基因表达分析
- 突变位点识别
- 疾病亚型分类

## 常用架构
- Attention MIL
- CLAM：数据高效MIL
- TransMIL：Transformer-based MIL
- DSMIL：双流MIL

## 优势
- 减少标注成本
- 适合大规模数据分析
- 弱监督场景效果好
- 可处理变长输入

## 局限性
- 实例标签不可知
- 对噪声实例敏感
- 聚合函数设计关键

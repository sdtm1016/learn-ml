# 3D-CNN（三维卷积神经网络）

## 简介
3D-CNN是卷积神经网络的三维扩展版本，通过三维卷积核处理体积数据，能够同时捕获空间和时间（或深度）维度的特征。

## 核心原理
- **三维卷积操作**：在三个维度上滑动卷积核
- **时空特征提取**：同时捕获空间结构和时序信息
- **体积数据处理**：直接处理三维医学影像数据

## 与2D-CNN的区别
- 卷积核从二维扩展到三维
- 输出为三维特征图
- 参数量更大，计算量更高
- 能捕获切片间的上下文信息

## 医疗应用场景

### 三维医学影像分析
- CT/MRI病灶检测
- 肺结节三维分割
- 脑肿瘤体积计算
- 肝脏肿瘤分割

### 视频分析
- 手术视频智能分析
- 超声心动图分析
- 内镜视频实时识别

### 时序医学数据
- 心电图时序分析
- 脑电图信号处理
- 生命体征监测

## 常用架构
- 3D U-Net
- V-Net
- 3D ResNet
- SlowFast Network

## 优势
- 充分利用三维空间信息
- 捕获切片间上下文关系
- 提高分割和检测精度

## 局限性
- 计算资源需求大
- 训练时间长
- 需要三维标注数据
- 显存占用高

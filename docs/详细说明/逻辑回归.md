# 逻辑回归（Logistic Regression）

## 简介
逻辑回归是一种经典的二分类算法，通过sigmoid函数将线性回归结果映射到0-1区间，输出概率值。因其可解释性强、计算简单，在医疗领域广泛应用。

## 核心原理
- **线性组合**：z = w·x + b
- **Sigmoid函数**：p = 1/(1+e^(-z))
- **损失函数**：交叉熵损失
- **正则化**：L1/L2正则化防止过拟合

## 主要变体
- 多项逻辑回归（多分类）
- 正则化逻辑回归
- 贝叶斯逻辑回归
- 加权逻辑回归

## 医疗应用场景

### 疾病风险预测
- 糖尿病风险预测
- 高血压风险预测
- 心血管疾病风险
- 脑卒中风险

### 诊断预测
- 脓毒症诊断
- 肿瘤良恶性判断
- 感染判断
- 器官功能评估

### 预后评估
- 死亡风险预测
- 复发风险
- 并发症预测

### 临床评分模型
- qSOFA评分
- CHA2DS2-VASc评分
- Wells评分

## 优势
- 可解释性强
- 计算效率高
- 输出概率值
- 特征重要性直观

## 局限性
- 只能处理线性可分问题
- 对多重共线性敏感
- 需要特征工程
- 处理非线性关系能力有限

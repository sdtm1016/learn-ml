# 生存分析（Survival Analysis）

## 简介
生存分析是研究事件发生时间的统计分析方法，广泛应用于医学研究中的预后预测、疗效评估、风险因素分析等。

## 核心概念
- **生存时间**：从起点到事件发生的时间
- **删失数据**：未观察到事件发生的数据
- **生存函数**：S(t) = P(T > t)
- **风险函数**：h(t) = 事件瞬时发生率

## 主要方法
- **Kaplan-Meier估计**：非参数生存曲线估计
- **Log-rank检验**：比较两组生存曲线
- **Cox比例风险模型**：半参数回归模型
- **参数模型**：Weibull、指数分布等

## 医疗应用场景

### 预后预测
- 肿瘤生存期预测
- 疾病复发时间预测
- 器官移植存活预测
- 慢性病进展预测

### 疗效评估
- 药物疗效比较
- 治疗方案选择
- 临床试验分析

### 风险因素分析
- 疾病危险因素识别
- 预后因素分析
- 生物标志物评估

### 神经退行性疾病
- 阿尔茨海默症进展预测
- 帕金森病进展预测

## 优势
- 处理删失数据
- 考虑时间因素
- 临床意义明确
- 可视化直观

## 局限性
- 假设检验条件
- 对样本量要求高
- 需要专业统计知识
- 模型假设可能不满足

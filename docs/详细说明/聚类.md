# 聚类（Clustering）

## 简介
聚类是一种无监督学习方法，将相似的数据点分组到同一簇中。在医疗领域用于患者分层、疾病亚型发现、基因表达分析等。

## 主要方法
- **划分聚类**：K-Means、K-Medoids
- **层次聚类**：凝聚式、分裂式
- **密度聚类**：DBSCAN、OPTICS
- **模型聚类**：高斯混合模型（GMM）

## 常用算法
- K-Means：经典划分聚类
- DBSCAN：基于密度的聚类
- 层次聚类：树状结构
- Spectral Clustering：谱聚类

## 评估指标
- 轮廓系数（Silhouette）
- Calinski-Harabasz指数
- Davies-Bouldin指数
- 肘部法则

## 医疗应用场景

### 患者分层
- 慢病患者分层管理
- 高血压患者分层
- 糖尿病患者分层
- 风险分层

### 疾病亚型发现
- 肿瘤亚型分类
- 糖尿病亚型识别
- 自身免疫病分型
- 精神疾病分型

### 基因表达分析
- 基因聚类
- 样本聚类
- 共表达基因发现

### 医学影像分析
- 病灶分割
- 组织分类
- 图像检索

### 用户画像
- 患者行为分析
- 就诊模式分析
- 健康管理分组

## 优势
- 无需标注数据
- 发现数据内在结构
- 支持探索性分析
- 可视化直观

## 局限性
- 聚类数目难以确定
- 对初始值敏感
- 对噪声和异常值敏感
- 不同算法结果差异大

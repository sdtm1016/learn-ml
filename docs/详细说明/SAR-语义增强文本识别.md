# SAR（语义增强文本识别）

## 简介
SAR（Show, Attend and Read: A Simple and Strong Baseline for Irregular Text Recognition）是一种基于语义增强的文本识别模型，由旷视科技提出。SAR采用简洁而有效的架构，通过注意力机制和语义增强策略实现高精度的文本识别，特别擅长处理不规则形状的文本。

## 模型架构
- **编码器**：CNN特征提取
- **注意力解码器**：带注意力的RNN序列解码
- **语义增强模块**：融合字符级语义信息

```
图像 → CNN编码器 → 特征序列 → 注意力解码器 + 语义增强 → 字符序列
```

## 核心原理

### CNN编码器
- 使用轻量级CNN（如ResNet-31）提取视觉特征
- 输出特征图序列
- 高效的特征提取

### 注意力解码器
- 使用GRU作为解码器
- 多头注意力机制（可选）
- 逐字符生成识别结果

### 语义增强
- 在解码过程中融合字符级语义信息
- 利用已识别的字符上下文增强后续字符预测
- 提升对模糊/低质量文本的识别能力

## 数学公式

### 编码器
```
H = CNN(image)    # H: H' × W' × D
```

### 注意力解码
```
s_t = GRU(y_{t-1}, s_{t-1}, c_t)
c_t = Σ_i α_{t,i} · h_i
```

### 语义增强
```
s_t_enhanced = GRU(s_t, context_prev)
context_prev = f(y_1, y_2, ..., y_{t-1})
```
- 利用前t-1步的字符预测作为语义上下文
- 增强当前步骤的预测能力

### 损失函数
```
L = -Σ_t log P(y_t | y_{<t}, image)
```

## 关键设计

| 设计 | 说明 | 作用 |
|------|------|------|
| 轻量编码器 | ResNet-31 | 快速特征提取 |
| GRU解码 | 替代LSTM | 更快，参数更少 |
| 多头注意力 | 并行多个注意力头 | 捕获多种对齐模式 |
| 语义增强 | 字符上下文融合 | 提升模糊文本识别 |
| 标签平滑 | 正则化 | 提高泛化能力 |

## 医疗应用场景

### 高精度OCR识别
- 医疗文档精确OCR
- 处方药物名称识别
- 检查报告数值精确提取
- 病理报告文本识别

### 不规则文本处理
- 拍照病历OCR
- 弯曲排列的医疗文档
- 低质量扫描文档识别
- 手写+打印混合文本

### 脑卒中专病数据采集
- 脑卒中病历高精度OCR
- 影像报告数值提取
- 检验结果精确识别
- 多种质量病历批量处理

## 优势
- 语义增强提升模糊文本识别
- 架构简洁，易于实现和训练
- 识别精度高
- 支持不规则文本
- 训练收敛快

## 局限性
- 对严重变形文本效果有限
- 无几何校正模块（需配合检测器）
- 长文本序列解码速度慢
- 需要大量训练数据
- 中文大字符集训练难度大

## 实现框架
- mmocr
- PyTorch
- PaddleOCR

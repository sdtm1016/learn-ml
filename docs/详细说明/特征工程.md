# 特征工程（Feature Engineering）

## 简介
特征工程是利用领域知识从原始数据中提取、构建和选择特征的过程，是机器学习成功的关键。在医疗领域需要结合医学专业知识进行特征设计。

## 主要任务
- **特征提取**：从原始数据提取特征
- **特征构建**：创建新特征
- **特征选择**：选择最有用的特征
- **特征变换**：标准化、归一化等

## 特征类型
- **数值特征**：连续型变量
- **类别特征**：离散型变量
- **时间特征**：时间相关特征
- **文本特征**：文本数据特征

## 医疗应用场景

### 检验指标特征
- 实验室检验值
- 生命体征
- 评分量表
- 变化趋势特征

### 影像特征
- 形态学特征
- 纹理特征
- 放射组学特征
- 深度学习特征

### 时序特征
- 时间窗口统计
- 变化率特征
- 周期性特征
- 趋势特征

### 文本特征
- TF-IDF特征
- 词嵌入特征
- 医学实体特征
- 主题特征

### 特征选择方法
- 过滤法：方差阈值、相关系数
- 包装法：递归特征消除
- 嵌入法：L1正则化、树模型重要性

## 优势
- 提高模型性能
- 减少计算开销
- 提高可解释性
- 结合领域知识

## 局限性
- 需要领域知识
- 耗时耗力
- 可能引入偏差
- 难以自动化

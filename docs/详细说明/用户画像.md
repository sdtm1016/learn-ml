# 用户画像（User Profiling）

## 简介
用户画像是通过分析用户行为、属性等数据构建的用户特征标签体系。在医疗领域用于患者管理、个性化服务、精准营销等。

## 核心要素
- **人口属性**：年龄、性别、地区等
- **健康属性**：疾病史、用药史、过敏史等
- **行为属性**：就诊行为、用药依从性等
- **偏好属性**：就医偏好、沟通偏好等

## 构建方法
- **标签体系构建**：定义标签类别
- **特征提取**：从数据中提取特征
- **标签计算**：计算用户标签值
- **画像更新**：动态更新画像

## 常用技术
- 聚类分析
- 关联规则
- 分类模型
- 知识图谱

## 医疗应用场景

### 慢病管理
- 高血压患者画像
- 糖尿病患者画像
- 慢病患者分层管理
- 个性化随访

### 健康管理
- 健康风险评估
- 生活方式分析
- 健康干预推荐

### 患者服务
- 个性化服务推荐
- 就诊提醒
- 健康教育推送

### 医院运营
- 患者流失预警
- 复诊预测
- 服务质量评估

## 优势
- 精准了解用户
- 支持个性化服务
- 提高运营效率
- 辅助决策制定

## 局限性
- 数据质量依赖
- 隐私保护要求
- 标签体系维护
- 动态更新困难

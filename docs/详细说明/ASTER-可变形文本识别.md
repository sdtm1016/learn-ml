# ASTER（可变形文本识别）

## 简介
ASTER（An End-to-End Scene Text Recognition with Focused Attention and Rectified Network）是一种可变形场景文本识别模型，由华中科技大学提出。ASTER通过可变形卷积网络（STN）对输入文本图像进行几何校正，然后使用带聚焦注意力的序列解码器进行文本识别，特别擅长处理弯曲、透视变形的文本。

## 模型架构
- **空间变换网络（STN）**：文本图像几何校正
- **修正网络（Rectified Network）**：基于CNN的特征提取
- **聚焦注意力解码器**：带注意力的序列解码

```
图像 → STN几何校正 → 修正CNN → 聚焦注意力解码器 → 文本
```

## 核心原理

### 空间变换网络（STN）
- **定位网络**：预测变换参数
- **网格生成器**：根据参数生成采样网格
- **采样器**：对输入图像进行几何变换
- 将弯曲/透视文本校正为水平文本

### 变换类型
- **TPS（薄板样条变换）**：处理任意弯曲变形
- **仿射变换**：处理旋转、缩放、平移
- **透视变换**：处理透视变形

### 聚焦注意力机制
- 在注意力计算中聚焦于文本区域
- 抑制背景噪声的注意力权重
- 使用门控机制控制注意力范围

## 数学公式

### TPS变换
```
TPS: (x, y) → (x', y')
```
- 通过控制点和TPS核函数实现平滑变换
- 支持任意形状的几何校正

### 注意力计算
```
e_{t,i} = W_g ⊙ a(s_{t-1}, h_i) + b_g ⊙ h_i
α_{t,i} = softmax(e_{t,i})
```
- W_g：门控权重
- ⊙：逐元素乘法
- 门控机制抑制无关区域

### 损失函数
```
L = L_rec + λ * L_STN
```
- L_rec：识别交叉熵损失
- L_STN：STN变换损失

## 医疗应用场景

### 变形文本识别
- 弯曲排列的病历文本识别
- 透视变形的扫描文档OCR
- 拍照倾斜的处方识别
- 圆柱面标签文字识别

### 医学影像标注
- 影像上弯曲文字标注识别
- 造影图像上的文字读取
- 超声影像标注识别

### 脑卒中专病数据采集
- 拍照脑卒中病历OCR
- 弯曲排版的检查报告识别
- 手机拍摄处方文字识别
- 非标准扫描文档OCR

## 优势
- STN几何校正能力强，处理弯曲/透视文本
- 聚焦注意力抑制背景噪声
- 端到端训练
- 对不规则文本识别效果好
- 支持多种几何变换

## 局限性
- STN增加计算开销
- 对极不规则文本校正有限
- 训练数据需要变换标注
- 复杂场景下STN可能校正失败
- 推理速度较慢

## 实现框架
- mmocr
- PyTorch
- PaddleOCR（部分集成）

## 与其他模型对比

| 模型 | 几何校正 | 弯曲文本 | 速度 | 准确率 |
|------|---------|---------|------|--------|
| CRNN | 无 | 差 | 最快 | 高 |
| Attention-OCR | 无 | 中 | 中 | 高 |
| **ASTER** | **STN+TPS** | **好** | 中 | 高 |
| SAR | 无 | 中 | 中 | 更高 |
| RARE | STN+仿射 | 良好 | 中 | 高 |
| STAR-Net | STN+仿射 | 良好 | 中 | 高 |

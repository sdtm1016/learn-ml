# ClinicalBERT-CRF（临床文本实体识别）

## 简介
ClinicalBERT-CRF是基于ClinicalBERT预训练语言模型和CRF（条件随机场）的命名实体识别模型。ClinicalBERT在临床文本（电子病历、临床笔记、出院摘要等）上进行预训练，特别擅长处理临床场景下的非结构化文本。

## 模型架构
- **ClinicalBERT编码层**：临床领域预训练的Transformer编码器
- **全连接层**：将BERT输出映射到标签空间
- **CRF层**：全局最优标签序列解码

```
输入文本 → ClinicalBERT编码器 → 全连接层 → CRF层 → 标签序列
```

## 核心原理

### ClinicalBERT预训练
- 在大规模临床语料上继续预训练BERT
- 使用MIMIC-III临床数据库（超过200万条临床笔记）
- 学习临床文本的语言特征和医学术语表示

### 临床文本特征
- 理解临床笔记的缩写和特殊表达
- 处理不规范的书写和术语变体
- 识别临床文档的结构化信息

### CRF层
- 标签转移概率建模
- 保证标签序列合法性
- Viterbi算法全局最优解码

## 与BioBERT的区别

| 特征 | ClinicalBERT | BioBERT |
|------|-------------|---------|
| 预训练语料 | 临床笔记、病历 | PubMed论文、PMC全文 |
| 文本风格 | 非正式、缩写多 | 学术规范、术语标准 |
| 适用场景 | 电子病历处理 | 文献挖掘 |
| 中文支持 | 需额外适配 | 需额外适配 |
| 实体侧重 | 临床发现、诊断、用药 | 基因、蛋白质、疾病 |

## 医疗实体识别类型

| 实体类型 | 临床文本示例 | 说明 |
|----------|-------------|------|
| 问题/诊断 | "患者突发右侧肢体无力" | 临床诊断 |
| 治疗 | "予阿替普酶溶栓治疗" | 治疗措施 |
| 检查 | "头颅CT示左侧基底节区梗死" | 检查及结果 |

| 严重程度 | "NIHSS评分12分" | 病情评估 |

## 优势
- 专为临床文本设计，理解临床表达习惯
- 对非正式缩写和临床术语处理能力强
- 在多个临床NER基准上达到SOTA（94-97%）
- 适合电子病历信息抽取
- CRF层保证标签序列合法性

## 局限性
- 依赖英文临床语料（MIMIC-III）
- 中文临床文本需重新预训练或微调
- 临床数据隐私要求高
- 模型参数量大，推理开销大
- 对扫描件/OCR文本效果下降

## 训练与微调

### 预训练数据
- MIMIC-III临床笔记
- 临床出院摘要
- 放射学报告

### 微调策略
- 全参数微调：数据充足时使用
- LoRA微调：资源受限时使用
- P-Tuning：少样本场景

### 标注方案
- BIO标注：B-Entity, I-Entity, O
- BIOES标注：增加边界标记
- 实体嵌套处理：嵌套NER方案

## 实现框架
- Hugging Face Transformers
- PyTorch
- TensorFlow

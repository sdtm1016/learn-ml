# NLP信息抽取（Information Extraction）

## 简介
NLP信息抽取是从非结构化文本中提取结构化信息的技术，在医疗领域用于电子病历结构化、医学文献挖掘、知识图谱构建等。

## 主要任务
- **命名实体识别（NER）**：识别文本中的实体
- **关系抽取**：识别实体间关系
- **事件抽取**：识别事件及参与者
- **属性抽取**：提取实体属性

## 常用方法
- **规则方法**：正则表达式、词典匹配
- **机器学习**：CRF、SVM
- **深度学习**：BiLSTM-CRF、BERT
- **预训练模型**：BioBERT、ClinicalBERT

## 医疗应用场景

### 电子病历结构化
- 症状提取
- 诊断提取
- 药物信息提取
- 手术史提取
- 过敏史提取
- 家族史提取

### 医学文献挖掘
- 研究结论提取
- 方法学信息提取
- 数据提取
- 关键发现抽取

### 知识图谱构建
- 实体抽取
- 关系抽取
- 属性填充
- 知识补全

### 自动生成院前电子急救病历
- 患者信息提取
- 病情描述结构化
- 处置记录提取
- 时间线构建

## 优势
- 自动化信息提取
- 结构化非结构化数据
- 提高数据利用率
- 支持后续分析

## 局限性
- 医学文本复杂性
- 标注数据获取难
- 实体歧义问题
- 领域适应性差

# Jieba（中文分词）

## 简介
Jieba（结巴分词）是Python中最流行的中文分词工具，由Sun Junyi开发。Jieba支持三种分词模式（精确模式、全模式、搜索引擎模式），结合基于词典的前缀匹配和HMM（隐马尔可夫模型）未登录词识别，能够高效准确地完成中文文本分词。

## 核心原理

### 基于前缀词典的分词
- 加载预定义词典，构建前缀词典（Trie树）
- 使用基于词典的最大匹配算法
- 支持动态添加词典

### 三种分词模式

| 模式 | 说明 | 示例 |
|------|------|------|
| 精确模式 | 最准确的分词 | "脑卒中" → "脑卒中" |
| 全模式 | 列出所有可能分词 | "脑卒中" → "脑/卒中/脑卒中" |
| 搜索引擎模式 | 精确模式+长词再切 | "脑卒中" → "脑/卒中" |

### HMM未登录词识别
- 对不在词典中的词使用HMM模型识别
- 四种标签：B（词首）、M（词中）、E（词尾）、S（单字词）
- 使用Viterbi算法解码最优标签序列

### 前缀匹配算法
- 基于Trie树的前缀匹配
- 从右向左或从左向右扫描
- 动态规划计算最优路径

## 数学公式

### HMM模型
```
状态集合: Q = {B, M, E, S}
观测集合: O = {汉字}
转移概率: A = P(q_{t+1} | q_t)
发射概率: B = P(o_t | q_t)
初始概率: π = P(q_1)
```

### Viterbi解码
```
δ_t(j) = max_i [δ_{t-1}(i) × A_{ij}] × B_j(o_t)
ψ_t(j) = argmax_i [δ_{t-1}(i) × A_{ij}]
```

### 词典匹配得分
```
score(path) = Σ log P(word) + λ × P(HMM_score)
```

## 优势
- 中文分词效果好
- 支持自定义词典
- 三种分词模式适应不同场景
- HMM未登录词识别
- 安装简单，使用方便
- 社区活跃，文档完善
- 支持并行分词

## 局限性
- 基于词典+规则，缺乏深度语义理解
- 对新词/专业术语需要手动添加词典
- 分词粒度不够灵活
- 无法处理多义词
- 与现代子词分词（BPE/WordPiece）不兼容
- 不适合作为大模型分词器

## 实现框架
- jieba（官方Python包）
- pkuseg（北大分词）
- LAC（百度中文分词）

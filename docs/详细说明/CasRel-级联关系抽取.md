# CasRel（级联关系抽取）

## 简介
CasRel（Cascaded Binary Tagging Framework）是一种级联二元标注框架，用于联合实体识别和关系抽取。由中国科学院深圳先进技术研究院提出，首次在重叠三元组抽取任务上取得了突破性进展，能够处理单实体对应多种关系（SPO重叠）的复杂场景。

## 模型架构
- **BERT编码层**：获取上下文感知的文本表示
- **主体（Subject）标注器**：识别文本中的所有主体实体
- **关系特定客体（Object）标注器**：针对每种关系，识别对应的客体实体

```
输入文本 → BERT编码器 → 主体标注器 → 关系特定客体标注器 → 三元组集合
                                    ↓
                            (subject, relation, object)
```

## 核心原理

### 级联框架
1. **第一阶段**：使用二元序列标注识别所有可能的主体实体
2. **第二阶段**：对每个识别出的主体，在每种关系下独立标注客体实体
3. 级联方式避免了传统流水线方法的错误传播

### 二元标注策略
- 每个token预测两个标签：序列开始（B）、序列内部（I）
- 主体标注：检测subject的起始和结束位置
- 客体标注：在给定subject和relation的条件下检测object

### 重叠三元组处理
- **EntityPairOverlap（EPO）**：同一实体对有多个关系
- **SingleEntityOverlap（SEO）**：一个实体同时作为多个三元组的主体或客体
- CasRel天然支持处理所有类型的重叠问题

## 数学公式

### 主体标注
```
p_sub(j) = sigmoid(W_s · h_j + b_s)    # subject起始位置
p_sub_e(j) = sigmoid(W_e · h_j + b_e)  # subject结束位置
```

### 客体标注（给定subject s和关系r）
```
p_obj_r(j | s) = sigmoid(W_r · [h_j; h_s_start; h_s_end] + b_r)
```

### 训练损失
```
L = L_sub + Σ_r L_obj_r
L_sub = -Σ_j [y_j^s log p_sub(j) + y_j^e log p_sub_e(j)]
```

- 治疗方案关联

### 药物知识图谱构建
- 药物-适应症
- 药物-禁忌症
- 药物-不良反应
- 药物-药物相互作用

### 脑卒中专病数据采集
- 脑卒中-危险因素关联
- 脑卒中-治疗方案关联

- 影像-诊断关系

## 优势
- 联合实体和关系抽取，避免错误传播
- 天然支持重叠三元组抽取
- 级联框架结构清晰
- 在多个基准数据集上达到SOTA
- 端到端训练

## 局限性
- 客体标注器数量随关系类型增加而增加
- 关系类型多时训练效率下降
- 对长文本效果有限
- 推理速度受关系类型数量影响

# DBNet（可微分二值化文本检测）

## 简介
DBNet（Differentiable Binarization Network）是一种基于可微分二值化的文本检测算法，由华中科技大学提出。通过将二值化过程纳入神经网络训练，实现了可微分的后处理操作，使得文本检测模型端到端可训练，在多个文本检测基准上达到SOTA。

## 模型架构
- **骨干网络**：ResNet特征提取
- **特征金字塔（FPN）**：多尺度特征融合
- **增强型特征融合**：级联特征融合
- **预测头**：概率图+阈值图
- **可微分二值化**：自适应阈值分割

```
图像 → ResNet → FPN → 特征融合 → 概率图 + 阈值图 → 可微分二值化 → 文本框
```

## 核心原理

### 可微分二值化（DB）
传统二值化使用固定阈值，DBNet将阈值也作为网络输出，使二值化过程可微分：
```
B_{i,j} = 1 / (1 + exp(-k * (P_{i,j} - T_{i,j})))
```
- P_{i,j}：概率图（文本区域概率）
- T_{i,j}：阈值图（自适应阈值）
- k：放大因子（经验值设为50）

### 自适应阈值
- 阈值图由网络学习得到
- 不同区域使用不同阈值
- 对光照变化、背景复杂等场景更鲁棒

### 后处理
- 从二值化结果中提取连通区域
- 使用Vatti clipping算法收缩边界
- 计算最小外接矩形作为文本框

## 数学公式

### 损失函数
```
L = L_s + α * L_b + β * L_t
```
- L_s：概率图二值交叉熵损失
- L_b：二值化图损失
- L_t：阈值图L1损失
- α, β：平衡系数

### 概率图损失
```
L_s = BCELoss(P, GT)
```

### 阈值图损失
```
L_t = Σ |T_{i,j} - GT_{i,j}|    (仅计算文本区域)
```

## 医疗应用场景

### 扫描病历OCR
- 纸质病历扫描件文本检测
- 手写处方文本定位
- 检查报告文本区域检测

### 医学影像标注检测
- 影像报告中的文本定位
- 医学文档中的表头识别
- 病理报告文本区域检测

### 脑卒中专病数据采集
- 扫描版脑卒中病历文本检测
- 影像学报告文本区域定位
- 手写病历文本识别预处理

## 优势
- 端到端可训练，无需复杂后处理
- 自适应阈值，鲁棒性强
- 检测精度高，速度快
- 支持弯曲文本检测（DBNet++）
- 适合工业部署

## 局限性
- 对极小文本检测效果有限
- 多语言混合场景需优化
- 训练数据标注成本高

## 变体

| 变体 | 特点 | 速度 | 精度 |
|------|------|------|------|
| DBNet | 基础版本 | 快 | 高 |
| DBNet++ | 增强版，弯曲文本 | 中 | 更高 |
| DBNet-ResNet50 | 轻量版 | 最快 | 中 |
| DBNet-ResNet152 | 高精度版 | 慢 | 最高 |

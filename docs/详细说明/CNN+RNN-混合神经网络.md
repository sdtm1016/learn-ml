# CNN+RNN（混合神经网络）

## 简介
CNN+RNN是将卷积神经网络和循环神经网络结合的混合架构，CNN负责提取空间特征，RNN负责建模时序依赖。在医疗领域用于信号分析、视频理解等场景。

## 架构设计
- **CNN部分**：提取空间/频域特征
- **RNN部分**：建模时序依赖关系
- **融合方式**：串联、并联、注意力融合

## 常见组合
- CNN + LSTM
- CNN + GRU
- CNN + BiLSTM
- ResNet + LSTM

## 医疗应用场景

### 信号分析
- EEG脑电图分析
- ECG心电图分析
- 睡眠分期
- 癫痫检测
- 心律失常识别

### 视频分析
- 手术视频理解
- 超声心动图分析
- 内镜视频分析
- 步态分析

### 多模态分析
- 影像+时序数据融合
- 语音+视觉融合
- 生命体征+影像融合

### ICU监测
- 多参数生命体征分析
- 病情恶化预警
- 呼吸模式分析

## 优势
- 同时捕获空间和时序特征
- 端到端学习
- 特征自动提取
- 适合多模态数据

## 局限性
- 计算复杂度高
- 训练时间长
- 需要大量数据
- 超参数调优复杂

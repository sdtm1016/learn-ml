# 因果推断（Causal Inference）

## 简介
因果推断是从观察数据中推断因果关系的方法，超越相关性分析，识别真正的因果效应。在医疗领域用于药物效果评估、治疗方案选择、危险因素分析等。

## 核心概念
- **因果效应**：干预对结果的因果影响
- **反事实**：未发生情况下的潜在结果
- **混杂因素**：同时影响干预和结果的因素
- **倾向评分**：接受干预的概率

## 主要方法
- **随机对照试验（RCT）**：金标准
- **倾向评分匹配（PSM）**：观察数据因果推断
- **工具变量（IV）**：解决内生性问题
- **双重差分（DID）**：面板数据因果推断

## 医疗应用场景

### 治疗效果评估
- 药物疗效评估
- 治疗方案比较
- 手术效果评估
- 干预措施评价

### 个性化治疗推荐
- 肿瘤化疗方案选择
- 靶向药选择
- 抗凝药剂量推荐
- 治疗方案优化

### 危险因素分析
- 疾病危险因素识别
- 病因学研究
- 流行病学调查

### 临床决策支持
- 治疗决策辅助
- 预后因素分析
- 并发症风险预测

## 优势
- 识别因果关系
- 支持决策制定
- 控制混杂因素
- 科学评估效果

## 局限性
- 需要强假设
- 观察数据偏差
- 未观测混杂因素
- 方法选择复杂

# SEIR模型（传染病模型）

## 简介
SEIR模型是一种经典的传染病动力学模型，将人群分为易感者(S)、暴露者(E)、感染者(I)和康复者(R)四类，用于模拟传染病传播过程。

## 模型结构
- **S（Susceptible）**：易感者，未感染但可能被感染
- **E（Exposed）**：暴露者，已感染但未发病
- **I（Infected）**：感染者，具有传染性
- **R（Recovered）**：康复者，已康复并获得免疫

## 核心参数
- β：传染率
- σ：潜伏期转化率
- γ：康复率
- R0：基本再生数

## 模型变体
- SIR模型：无潜伏期
- SEIRS模型：免疫可消失
- SEIRD模型：考虑死亡
- 年龄结构SEIR：分年龄组

# EfficientNet（高效网络）

## 简介
EfficientNet由Google在2019年提出，通过复合缩放方法同时优化网络的深度、宽度和分辨率，在保持高精度的同时大幅减少参数量和计算量。

## 核心思想
- **复合缩放（Compound Scaling）**：统一缩放深度、宽度、分辨率
- **基准网络（Baseline）**：EfficientNet-B0
- **缩放系数**：通过系数φ控制整体缩放程度

## 系列模型
- EfficientNet-B0 到 B7
- EfficientNetV2：改进训练速度
- EfficientNet-EdgeTPU：针对边缘设备优化

## 医疗应用场景

### X光胸片诊断
- 肺炎检测
- 多标签胸部疾病分类
- 结核筛查

### 移动端医疗应用
- 便携式超声设备
- 移动DR影像分析
- 远程医疗诊断

### 资源受限环境
- 边缘计算设备
- 嵌入式医疗设备
- 实时诊断系统

## 优势
- 参数效率高
- 计算速度快
- 精度与效率平衡好
- 适合部署到移动设备

## 局限性
- 训练时间较长
- 对数据增强敏感
- 某些变体显存占用较高

# BioBERT-CRF（生物医学实体识别）

## 简介
BioBERT-CRF是基于BioBERT预训练语言模型和CRF（条件随机场）的命名实体识别模型。BioBERT在大规模生物医学语料（PubMed摘要、PMC全文）上进行了预训练，能够更好地理解生物医学领域的专业术语和语义关系。

## 模型架构
- **BioBERT编码层**：生物医学领域预训练的Transformer编码器
- **全连接层**：将BERT输出映射到标签空间
- **CRF层**：全局最优标签序列解码

```
输入文本 → BioBERT编码器 → 全连接层 → CRF层 → 标签序列
```

## 核心原理

### BioBERT预训练
- 在BERT基础上，使用PubMed（1400万+摘要）和PMC（27万+全文）继续预训练
- 保留BERT原有的MLM和NSP任务
- 学习生物医学领域的语言模式和术语表示

### 领域适应性
- 掌握医学术语、缩写、基因名、药物名等
- 理解生物医学文本的句法结构
- 捕获医学概念间的语义关联

### CRF层
- 建模标签间的转移概率
- 确保标签序列的合法性
- Viterbi算法解码

## 医疗实体识别类型

| 实体类型 | 示例 | 标准编码 |
|----------|------|----------|
| 疾病/障碍 | 脑卒中、心肌梗死 | ICD-10 |
| 症状/体征 | 偏瘫、言语不清 | SNOMED CT |
| 药物/化学物质 | 阿司匹林、tPA | ATC编码 |
| 基因/蛋白质 | APOE、BRCA1 | HGNC |
| 检查/检验 | CT、血常规 | LOINC |
| 解剖部位 | 大脑中动脉、基底节 | SNOMED CT |
| 手术操作 | 溶栓术、支架植入 | ICD-9-CM |

## 医疗应用场景

### 生物医学文献挖掘
- PubMed论文实体标注
- 基因-疾病关系抽取
- 药物-靶点关系提取
- 临床试验信息抽取

### 电子病历处理
- 出院小结实体识别
- 手术记录结构化
- 影像报告信息提取
- 病理科报告分析

### 脑卒中专病数据采集
- 脑卒中相关术语识别
- 危险因素标注
- 影像学发现提取
- 治疗方案结构化

## 评估指标

| 指标 | Entity-level | Token-level |
|------|-------------|-------------|
| Precision | 94%+ | 96%+ |
| Recall | 92%+ | 94%+ |
| F1-Score | 93-96% | 95-97% |

## 优势
- 生物医学领域预训练，术语理解能力强
- 在多个生物医学NER基准上达到SOTA
- 对基因名、药物名等专业实体识别准确
- 支持多种生物医学下游任务
- 可直接使用预训练权重微调

## 局限性
- 主要针对英文生物医学文本
- 中文场景需要额外适配
- 预训练语料偏重文献，临床病历需微调
- 模型参数量大，部署成本高

## 训练数据
- BC2GM（基因命名）
- BC5CDR（疾病/化学物质）
- JNLPBA（生物医学实体）
- NCBI-disease（疾病）
- ChemProt（化学物质-蛋白质关系）

## 实现框架
- Hugging Face Transformers
- BioBERT官方仓库
- PyTorch / TensorFlow

# 行为识别（Action Recognition）

## 简介
行为识别是从视频或传感器数据中识别特定动作或行为的技术，在医疗领域用于手术视频分析、康复训练评估、老年人跌倒检测等。

## 主要方法
- **基于骨架**：利用人体关节点信息
- **基于RGB视频**：直接从视频学习
- **基于光流**：利用运动信息
- **多模态融合**：结合多种数据源

## 常用模型
- 3D CNN（C3D、I3D）
- Two-Stream Network
- SlowFast Network
- Temporal Segment Network（TSN）
- Transformer-based方法

## 医疗应用场景

### 手术视频分析
- 手术步骤识别
- 器械识别
- 手术阶段划分
- 失误风险预警

### 康复训练
- 动作质量评估
- 康复训练指导
- 运动功能评估
- 步态分析

### 老年人监护
- 跌倒检测
- 异常行为识别
- 日常活动监测
- 安全预警

### 神经疾病评估
- 帕金森病运动评估
- 癫痫发作检测
- 运动障碍分析

## 优势
- 自动化行为分析
- 客观评估标准
- 实时监测能力
- 减轻人工负担

## 局限性
- 视角依赖性强
- 对遮挡敏感
- 计算资源需求高
- 复杂行为识别困难

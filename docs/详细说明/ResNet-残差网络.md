# ResNet（残差网络）

## 简介
ResNet（Residual Network）由微软研究院在2015年提出，通过引入残差连接解决了深层网络训练中的梯度消失问题，使网络可以训练数百甚至数千层。

## 核心创新
- **残差块（Residual Block）**：跳跃连接实现恒等映射
- **残差学习**：学习残差函数F(x) = H(x) - x，而非直接学习H(x)
- **批量归一化（Batch Normalization）**：加速训练，提高稳定性

## 主要变体
- ResNet-18/34/50/101/152
- ResNeXt：分组卷积
- DenseNet：密集连接
- Wide ResNet：加宽网络

## 医疗应用场景

### X光胸片诊断
- 肺炎检测
- 结核筛查
- 气胸识别
- 心影增大判断

### 医学影像分类
- 病灶良恶性分类
- 疾病多标签分类
- 影像质量评估

### 特征提取骨干网络
- 作为目标检测骨干网络
- 作为分割网络编码器
- 迁移学习预训练模型

## 优势
- 解决深层网络退化问题
- 训练更深的网络
- 特征提取能力强
- 广泛的预训练模型可用

## 局限性
- 参数量较大
- 计算量较高
- 对小目标检测效果有限

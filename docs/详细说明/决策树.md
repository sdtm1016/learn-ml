# 决策树（Decision Tree）

## 简介
决策树是一种基于树形结构进行决策的机器学习算法，通过一系列规则对数据进行分类或回归。因其可解释性强，在医疗领域广泛应用。

## 核心概念
- **根节点**：树的起始节点
- **内部节点**：决策节点
- **叶节点**：输出结果
- **分支**：决策路径

## 分裂准则
- **信息增益**：ID3算法
- **信息增益率**：C4.5算法
- **基尼指数**：CART算法

## 医疗应用场景

### 疾病诊断
- 疾病分类诊断
- 鉴别诊断辅助
- 症状分析
- 风险分层

### 预后预测
- 死亡风险预测
- 复发风险预测
- 并发症预测
- 康复预测

### 临床决策支持
- 治疗方案选择
- 检查项目推荐
- 用药决策辅助

### 检验指标分析
- 异常指标识别
- 检验结果解读
- 辅助诊断

## 优势
- 可解释性强
- 不需要特征缩放
- 处理非线性关系
- 自动特征选择

## 局限性
- 容易过拟合
- 对噪声敏感
- 不稳定性高
- 处理连续变量效果有限

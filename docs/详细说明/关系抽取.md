# 关系抽取（Relation Extraction）

## 简介
关系抽取是从文本中识别实体间语义关系的任务，是构建知识图谱的关键步骤。在医疗领域用于药物相互作用、疾病-症状关系、治疗方案关系等抽取。

## 主要方法
- **基于规则**：模板匹配、依存路径
- **监督学习**：分类模型
- **远程监督**：利用知识库自动标注
- **深度学习**：CNN、LSTM、BERT

## 常用模型
- PCNN：分段卷积网络
- BiLSTM-Attention
- BERT-based模型
- GNN-based模型

## 医疗应用场景

### 药物相互作用
- 药物-药物相互作用
- 药物-不良反应关系
- 药物-禁忌症关系
- 药物-适应症关系

### 疾病知识图谱
- 疾病-症状关系
- 疾病-病因关系
- 疾病-治疗关系
- 疾病-并发症关系

### 临床指南挖掘
- 诊断标准提取
- 治疗方案关系
- 推荐等级识别

### 罕见病诊断辅助
- 症状-疾病关系
- 基因-疾病关系
- 表型-疾病关系

## 优势
- 自动构建知识图谱
- 发现隐含关系
- 支持知识推理
- 提高信息利用率

## 局限性
- 关系类型定义困难
- 标注数据获取难
- 远程监督噪声
- 复杂关系抽取困难

# TextCNN（文本卷积神经网络）

## 简介
TextCNN是由Yoon Kim于2014年提出的一种用于文本分类的卷积神经网络模型，通过多尺度卷积核捕获不同粒度的n-gram特征，在文本分类任务中表现出色。该模型结构简洁、训练快速，适用于短文本分类场景。

## 模型架构
- **Embedding层**：将输入文本转换为词向量矩阵
- **卷积层**：多尺度一维卷积提取n-gram特征
- **池化层**：最大池化提取最重要的特征
- **全连接层**：分类输出

```
输入文本 → Embedding → 多尺度卷积(3,4,5) → 最大池化 → 拼接 → 全连接 → Softmax
```

## 核心原理

### 多尺度卷积
- 使用不同窗口大小的卷积核（如3、4、5）
- 每种窗口大小捕获不同n-gram特征
- 窗口大小为3捕获三词组合特征
- 窗口大小为5捕获五词组合特征

### 最大池化
- 对每个卷积核的输出取最大值
- 提取每个特征图中最显著的特征
- 实现变长文本的固定维度表示
- 增强模型对局部特征的鲁棒性

### 特征拼接
- 将不同卷积核的最大池化结果拼接
- 形成全局特征表示
- 送入全连接层进行分类

## 数学公式

### 卷积操作
```
c_i = ReLU(W · x_{i:i+h-1} + b)
```
- x_{i:i+h-1}：从位置i开始的h个词向量
- W：卷积核权重，h为窗口大小
- b：偏置项

### 最大池化
```
c_hat = max(c_1, c_2, ..., c_{n-h+1})
```

### 输出
```
y = Softmax(W_o · [c_hat_3; c_hat_4; c_hat_5] + b_o)
```

## 网络结构参数

| 参数 | 典型配置 | 说明 |
|------|---------|------|
| 词向量维度 | 128/256/300 | 每个词的嵌入维度 |
| 卷积核窗口 | 3, 4, 5 | 捕获不同n-gram |
| 每种窗口卷积核数 | 100/128/256 | 特征图数量 |
| Dropout率 | 0.5 | 防止过拟合 |
| 激活函数 | ReLU | 非线性变换 |

## 医疗应用场景

### 病历文本分类
- 自动区分病历类型（病史/诊断/检查报告/体格检查）
- 医疗文本意图识别
- 科室自动分类

### 医疗舆情分析
- 医疗新闻分类
- 患者评价情感分析
- 健康科普分类

### 脑卒中专病数据采集
- 病历文本类型自动分类
- 脑卒中相关文本筛选
- 信息类别判断

## 优势
- 结构简洁，易于实现
- 训练速度快，推理高效
- 多尺度卷积捕获丰富特征
- 对短文本分类效果好
- 参数量相对较少
- 支持GPU并行计算

## 局限性
- 缺乏长距离依赖建模能力
- 无法处理词序信息（池化后丢失位置信息）
- 对长文本效果有限
- 无法理解上下文语义（静态词向量）

## 实现框架
- PyTorch：nn.Conv1d + nn.MaxPool1d
- TensorFlow/Keras：Conv1D + GlobalMaxPooling1D
- FastText：内置TextCNN实现

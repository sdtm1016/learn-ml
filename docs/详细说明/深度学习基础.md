# 深度学习基础（Deep Learning Basics）

## 简介
深度学习是机器学习的分支，通过多层神经网络学习数据的层次化表示。在医疗领域广泛应用于影像分析、信号处理、自然语言处理等。

## 核心概念
- **神经网络**：模拟人脑的计算模型
- **前向传播**：输入到输出的计算过程
- **反向传播**：梯度计算和参数更新
- **激活函数**：引入非线性

## 常用层类型
- **全连接层**：Dense层
- **卷积层**：Conv层
- **池化层**：Pooling层
- **归一化层**：BatchNorm层
- **Dropout层**：防止过拟合

## 激活函数
- ReLU：f(x) = max(0, x)
- Sigmoid：f(x) = 1/(1+e^(-x))
- Tanh：f(x) = (e^x - e^(-x))/(e^x + e^(-x))
- Softmax：多分类输出

## 损失函数
- 交叉熵损失：分类任务
- 均方误差：回归任务
- Dice损失：分割任务
- Focal损失：类别不平衡

## 优化器
- SGD：随机梯度下降
- Adam：自适应学习率
- RMSprop：均方根传播
- AdamW：权重衰减Adam

## 医疗应用场景
- 医学影像分析
- 生物信号处理
- 电子病历分析
- 药物发现
- 基因组学

## 优势
- 自动特征学习
- 处理复杂模式
- 端到端学习
- 表现优异

## 局限性
- 需要大量数据
- 计算资源需求高
- 可解释性差
- 超参数调优复杂

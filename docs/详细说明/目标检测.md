# 目标检测（Object Detection）

## 简介
目标检测是识别图像中感兴趣目标的位置和类别的技术，结合了分类和定位任务。在医疗领域用于病灶检测、内镜实时识别、手术器械检测等。

## 主要类型
- **单阶段检测**：直接预测位置和类别
- **两阶段检测**：先生成候选区域再分类

## 经典模型
- **单阶段**：YOLO系列、SSD、RetinaNet
- **两阶段**：Faster R-CNN、Mask R-CNN、Cascade R-CNN
- **Transformer**：DETR、Deformable DETR

## 医疗应用场景

### 内镜实时检测
- 胃癌病灶检测
- 结直肠息肉识别
- 食管病变检测
- 出血点检测

### 医学影像检测
- 肺结节检测
- 骨折检测
- 肿瘤定位
- 钙化点检测

### 病理图像检测
- 细胞检测
- 腺体识别
- 癌细胞定位
- 有丝分裂检测

### 手术辅助
- 手术器械识别
- 手术步骤识别
- 术中实时导航

### 超声检测
- 甲状腺结节检测
- 乳腺病灶检测
- 胎儿结构检测

## 评估指标
- mAP（平均精度均值）
- AP50、AP75
- 精确率-召回率曲线
- FPS（每秒帧数）

## 优势
- 同时定位和分类
- 实时检测能力
- 多目标检测
- 支持临床决策

## 局限性
- 小目标检测困难
- 密集目标检测挑战
- 对遮挡敏感
- 计算资源需求高

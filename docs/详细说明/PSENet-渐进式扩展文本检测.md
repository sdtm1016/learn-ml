# PSENet（渐进式扩展文本检测）

## 简介
PSENet（Progressive Scale Expansion Network）是一种渐进式扩展的场景文本检测算法，由华中科技大学提出。通过生成不同尺度的文本核（text kernel），逐步扩展为完整的文本区域，有效解决了文本实例之间紧密相邻或交叉的问题。

## 模型架构
- **骨干网络**：ResNet特征提取
- **特征金字塔（FPN）**：多尺度特征融合
- **渐进式扩展模块**：生成多尺度文本核
- **后处理**：逐步扩展+连通区域分析

```
图像 → ResNet → FPN → 多尺度文本核预测 → 渐进式扩展 → 文本框
                              (S1, S2, S3, S4)
```

## 核心原理

### 渐进式扩展
1. **最小尺度核（S1）**：文本中心区域，不同文本实例分离
2. **逐步扩展（S2→S3→S4）**：逐渐扩大核的范围
3. **完整区域**：最终扩展为完整文本区域

### 文本核
- 最小核（S1）确保不同文本实例不重叠
- 逐步扩展过程中，同一实例的核逐步合并
- 不同实例的核保持分离

### 扩展算法
```
for each scale s in [S1, S2, S3, S4]:
    merge connected components in s
    expand to next scale
```

## 数学公式

### 多尺度预测
```
P = {P_1, P_2, ..., P_n}    # n个尺度的预测图
```

### 损失函数
```
L = Σ_{k=1}^{n} L_k
L_k = BCELoss(P_k, GT_k) + DiceLoss(P_k, GT_k)
```

### Dice损失
```
Dice = 2|P ∩ GT| / (|P| + |GT|)
L_dice = 1 - Dice
```

## 医疗应用场景

### 紧密排列文本检测
- 表格型检查报告文本检测
- 紧密排列的病历字段检测
- 多列文档版面分析

### 医疗文档OCR预处理
- 纸质病历扫描件文本区域定位
- 检验报告文本行检测
- 医学文档中的文本块分割

### 脑卒中专病数据采集
- 脑卒中病历表格文本检测
- 紧密排列的临床数据字段定位
- 扫描文档版面分析

## 优势
- 有效处理紧密相邻的文本实例
- 渐进式扩展策略准确可靠
- 对任意形状文本有较好支持
- 多尺度特征融合
- 基于分割的方法，无需锚框

## 局限性
- 后处理步骤相对复杂
- 扩展过程需要多次迭代
- 推理速度较慢
- 内存消耗较大（多尺度预测图）

## 变体

| 变体 | 特点 | 速度 | 精度 |
|------|------|------|------|
| PSENet-1s | 单尺度扩展 | 快 | 中 |
| **PSENet** | 多尺度渐进扩展 | 中 | 高 |
| PSENet-Lite | 轻量版 | 最快 | 中低 |

## 实现框架
- PaddleOCR
- mmocr
- PyTorch

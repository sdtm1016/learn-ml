# CNN（卷积神经网络）

## 简介
卷积神经网络（Convolutional Neural Network，CNN）是一种专门用于处理具有网格结构数据的深度学习模型，在图像识别、计算机视觉等领域取得了巨大成功。

## 核心组件
- **卷积层（Convolutional Layer）**：通过卷积核提取局部特征
- **池化层（Pooling Layer）**：降低特征维度，增强模型鲁棒性
- **全连接层（Fully Connected Layer）**：进行最终分类或回归

## 主要变体
- LeNet、AlexNet、VGG、ResNet、DenseNet
- 3D-CNN（处理视频、医学影像等三维数据）

## 医疗应用场景

### 医学影像诊断
- 肺结节检测与分类
- 脑肿瘤分割
- 眼底病变识别
- 乳腺癌筛查

### 病理图像分析
- 宫颈癌细胞学筛查
- 乳腺癌分级
- 前列腺癌诊断

### 内镜图像识别
- 胃癌早期筛查
- 结直肠息肉识别
- 食管病变检测

## 优势
- 自动特征提取，无需手工设计特征
- 参数共享机制，减少模型参数
- 平移不变性，对位置变化具有鲁棒性

## 局限性
- 需要大量标注数据
- 对旋转、缩放等变换敏感
- 计算资源需求较高

## 常用框架
- TensorFlow / Keras
- PyTorch
- MXNet

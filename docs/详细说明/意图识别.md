# 意图识别（Intent Recognition）

## 简介
意图识别是自然语言理解的核心任务，旨在识别用户输入的真实意图。在医疗领域用于智能问诊、导诊系统、患者咨询等场景。

## 核心任务
- **意图分类**：将用户输入分类到预定义意图类别
- **槽位填充**：提取意图相关的关键信息
- **多意图识别**：处理包含多个意图的输入

## 常用方法
- **规则匹配**：关键词、正则表达式
- **机器学习**：SVM、随机森林
- **深度学习**：CNN、LSTM、BERT
- **预训练模型**：意图分类微调

## 医疗应用场景

### 智能问诊
- 症状咨询意图
- 用药咨询意图
- 检查咨询意图
- 预约挂号意图

### 智能导诊
- 科室推荐
- 医生推荐
- 就诊建议

### 慢病管理
- 随访意图识别
- 健康咨询意图
- 用药提醒意图

### 患者服务
- 投诉建议意图
- 费用查询意图
- 报告查询意图

## 优势
- 提升用户体验
- 自动化服务流程
- 减少人工成本
- 24小时服务

## 局限性
- 意图定义困难
- 多意图处理复杂
- 口语化表达理解困难
- 需要持续优化

# YOLO（目标检测算法）

## 简介
YOLO（You Only Look Once）是一种实时目标检测算法，将目标检测任务转化为回归问题，实现端到端的快速检测。

## 核心思想
- **单阶段检测**：直接预测边界框和类别
- **网格划分**：将图像划分为S×S网格
- **端到端训练**：统一的目标检测框架
- **实时性能**：高FPS检测速度

## 版本演进
- YOLOv1-v3：基础版本
- YOLOv4-v5：性能优化
- YOLOv6-v8：架构改进
- YOLOv9-v10：最新发展

## 医疗应用场景

### 内镜实时检测
- 胃癌早期筛查
- 结直肠息肉识别
- 食管病变检测
- 息肉良恶性判断

### 医学影像检测
- 肺结节检测
- 骨折检测
- 病灶定位

### 手术辅助
- 手术器械识别
- 手术步骤识别
- 术中实时导航

## 优势
- 检测速度快
- 实时性好
- 端到端训练
- 部署简单

## 局限性
- 小目标检测效果有限
- 密集目标检测困难
- 定位精度相对较低
- 对遮挡敏感

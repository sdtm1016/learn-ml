# Word2Vec（词向量模型）

## 简介
Word2Vec由Google的Mikolov等人于2013年提出，是一种将词语映射为低维稠密向量的模型。Word2Vec通过无监督学习从大规模语料中学习词的语义表示，使得语义相近的词在向量空间中距离较近，是自然语言处理领域最具影响力的词向量方法之一。

## 核心架构

### CBOW（连续词袋模型）
- **目标**：通过上下文词预测中心词
- **输入**：中心词周围的上下文词
- **输出**：中心词的概率分布

```
上下文词 [w_{t-2}, w_{t-1}, w_{t+1}, w_{t+2}] → 平均/求和 → 隐藏层 → Softmax → 中心词 w_t
```

### Skip-gram（跳字模型）
- **目标**：通过中心词预测上下文词
- **输入**：中心词
- **输出**：上下文词的概率分布

```
中心词 w_t → 输入层 → 隐藏层 → Softmax → 上下文词 [w_{t-2}, w_{t-1}, w_{t+1}, w_{t+2}]
```

## 数学公式

### CBOW损失函数
```
L = -log P(w_t | w_{t-c}, ..., w_{t+c})
P(w_t | context) = softmax(h · W'_w_t)
```

### Skip-gram损失函数
```
L = -Σ_{-c≤j≤c, j≠0} log P(w_{t+j} | w_t)
P(w_{t+j} | w_t) = softmax(v_{w_t} · v'_{w_{t+j}})
```

### 负采样优化
```
L = -log σ(v'_c · v_t) - Σ_{k=1}^{K} log σ(-v'_k · v_t)
```
- σ：sigmoid函数
- K：负样本数量
- 大幅加速训练过程

### 负采样概率
```
P(w) = f(w)^0.75 / Σ_{w'} f(w')^0.75
```
- f(w)：词w的词频
- 0.75次方平滑低频词采样概率

## 核心特性

| 特性 | 说明 | 示例 |
|------|------|------|
| 语义相似 | 语义相近的词向量距离近 | king ≈ queen |
| 线性关系 | 向量空间中的线性关系 | king - man + woman ≈ queen |
| 类比推理 | 支持语义类比 | China - Beijing ≈ Japan - Tokyo |
| 聚类特性 | 同类词在向量空间中聚集 | 疾病词聚在一簇 |

## 医疗应用场景

### 医学词向量训练
- 从电子病历/医学文献训练领域词向量
- 医学实体的语义表示
- 医学术语的相似度计算

### 医疗文本处理
- 病历文本特征提取
- 医学实体相似度计算
- 医学知识图谱嵌入

### 脑卒中专病数据采集
- 脑卒中相关术语语义表示
- 医学术语相似度计算
- 术语归一化的基础特征
- 同义词发现

## 优势
- 训练速度快（负采样优化）
- 词向量质量高
- 支持大规模语料训练
- 预训练词向量可直接使用
- 向量维度可控（100-300维）
- 计算资源需求低

## 局限性
- 静态词向量（一词一义）
- 无法处理多义词
- 缺乏上下文理解
- 无法处理未登录词（OOV）
- 对低频词表示质量差

## 实现框架
- Gensim：Word2Vec实现
- TensorFlow：tf.keras.Embedding
- PyTorch：nn.Embedding

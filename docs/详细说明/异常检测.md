# 异常检测（Anomaly Detection）

## 简介
异常检测是识别与正常模式显著不同的数据点的技术，在医疗领域用于发现罕见疾病、异常指标、欺诈行为等。

## 主要方法
- **统计方法**：Z-score、IQR
- **基于距离**：KNN、LOF
- **基于密度**：DBSCAN、Isolation Forest
- **基于重构**：AutoEncoder
- **基于分类**：One-Class SVM

## 常用算法
- Isolation Forest（孤立森林）
- Local Outlier Factor（LOF）
- One-Class SVM
- AutoEncoder
- DBSCAN

## 医疗应用场景

### 检验指标异常
- 血常规异常识别
- 生化指标异常
- 感染判断
- 器官功能异常

### ICU监测
- 生命体征异常
- 早期病情恶化预警
- 设备故障检测
- 异常事件识别

### 医疗欺诈检测
- 医保欺诈识别
- 异常处方检测
- 异常就诊行为

### 罕见病筛查
- 罕见病识别
- 异常症状检测
- 遗传病筛查

## 优势
- 无需标注异常样本
- 可发现未知异常模式
- 适合不平衡数据

## 局限性
- 难以确定异常阈值
- 对正常模式定义敏感
- 可能产生大量误报
- 异常类型多样难以统一建模

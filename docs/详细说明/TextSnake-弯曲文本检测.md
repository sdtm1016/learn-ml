# TextSnake（弯曲文本检测）

## 简介
TextSnake是一种基于几何图元的弯曲文本检测算法，由华中科技大学提出。不同于传统的边界框检测方法，TextSnake将文本表示为一系列有序的圆盘（disk），每个圆盘有中心点、半径和方向，能够精确描述任意形状的弯曲文本。

## 模型架构
- **骨干网络**：ResNet/VGG特征提取
- **特征金字塔（FPN）**：多尺度特征融合
- **预测头**：预测文本区域、中心线、几何属性

```
图像 → 骨干网络 → FPN → 预测头 → 文本区域 + 中心线 + 几何属性 → TextSnake重建 → 文本框
```

## 核心原理

### TextSnake表示
每个文本实例表示为一组有序的圆盘序列：
```
T = {(C_i, R_i, cos θ_i, sin θ_i)}_{i=1}^{N}
```
- C_i：第i个圆盘的中心坐标
- R_i：第i个圆盘的半径（文本宽度的一半）
- θ_i：第i个圆盘的方向角

### 预测头输出
- **文本区域图（TR）**：文本/非文本分类
- **中心线图（TCL）**：文本中心线区域
- **半径图（R）**：每个中心线点到边界的距离
- **方向图（cos θ, sin θ）**：文本局部方向

### TextSnake重建
1. 从中心线图提取中心线点
2. 获取每个点的半径和方向
3. 按顺序连接圆盘，形成弯曲文本区域
4. 从圆盘序列计算多边形文本框

## 数学公式

### 圆盘表示
```
Disk_i = {(x, y) | ||(x, y) - C_i|| ≤ R_i}
```

### TextSnake区域
```
Snake(T) = ∪_{i=1}^{N} Disk_i
```

### 方向预测
```
cos θ = W_cos · h + b_cos
sin θ = W_sin · h + b_sin
```

### 损失函数
```
L = L_tr + λ_1 * L_tcl + λ_2 * L_r + λ_3 * L_θ
```
- L_tr：文本区域分类损失
- L_tcl：中心线分类损失
- L_r：半径回归损失
- L_θ：方向回归损失

## 医疗应用场景

### 弯曲文本检测
- 弯曲排列的医学文档文本
- 圆形/弧形标签文本
- 不规则排版的病历文本

### 医疗影像标注检测
- 医学影像中的弯曲文字标注
- 图像上的弧形测量标注
- 造影影像上的文字标记

### 脑卒中专病数据采集
- 扫描版弯曲排列病历文本检测
- 影像报告中的非规则文本区域
- 手写病历中的弯曲文本行

## 优势
- 支持任意形状的弯曲文本
- 精确的文本几何描述
- 对弧形、波浪形文本检测效果好
- 基于分割的方法，无需锚框
- 可重建精确的文本区域轮廓

## 局限性
- 对直线文本效果不如专门的水平/旋转检测器
- 推理速度较慢（中心线提取和重建复杂）
- 后处理步骤多
- 中心线预测需要高质量标注
- 对密集文本效果一般

## 实现框架
- mmocr
- PyTorch
- OpenCV

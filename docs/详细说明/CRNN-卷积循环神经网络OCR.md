# CRNN（卷积循环神经网络OCR）

## 简介
CRNN（Convolutional Recurrent Neural Network）是一种端到端的文本识别模型，由百度于2016年提出。结合CNN的图像特征提取能力和RNN的序列建模能力，通过CTC（Connectionist Temporal Classification）实现不定长文本识别，是OCR领域最经典的模型之一。

## 模型架构
- **CNN特征提取**：提取图像的视觉特征序列
- **RNN序列建模**：对特征序列进行上下文建模
- **CTC解码**：将序列输出解码为文本

```
图像 → CNN特征提取 → 特征序列 → RNN序列建模 → CTC解码 → 文本
```

## 核心原理

### CNN特征提取
- 使用VGG-like或ResNet结构提取图像特征
- 卷积层提取局部视觉特征
- 池化层降低空间维度
- 最终输出特征序列：宽度为W，高度压缩为1

### RNN序列建模
- 双向LSTM捕获特征序列的上下文依赖
- 前向LSTM从左到右读取特征
- 后向LSTM从右到左读取特征
- 输出每个时间步的字符概率分布

### CTC解码
- 处理不定长文本识别问题
- 允许输出中出现空白符（blank）和重复字符
- 通过CTC损失函数训练

## 数学公式

### CNN特征序列
```
f = CNN(image)    # 输出: C × H × W
```

### RNN序列建模
```
h_t = BiLSTM(f_t)
```

### CTC概率
```
P(l|x) = Σ_{π ∈ B^{-1}(l)} P(π|x)
```
- π：CTC路径（含blank和重复）
- B^{-1}(l)：文本l的所有可能CTC路径
- 通过前向-后向算法高效计算

### CTC损失
```
L_{CTC} = -log P(l|x)
```

### 解码策略
- **贪心解码**：每步取最大概率
- **束搜索（Beam Search）**：搜索最优路径

## 网络结构

| 组件 | 结构 | 输入 | 输出 |
|------|------|------|------|
| CNN | Conv+BN+ReLU+Pool | H×W×3 | 1×W'×C |
| RNN | BiLSTM (256×2) | W'×C | W'×512 |
| CTC | Linear + Softmax | W'×512 | W'×N_chars |

## 医疗应用场景

### 病历文本识别
- 纸质病历扫描件OCR识别
- 手写处方识别
- 检查报告文本识别
- 出院小结OCR

### 医学文档数字化
- 历史病历数字化归档
- 医学文献OCR
- 医疗文档信息提取

### 脑卒中专病数据采集
- 扫描版脑卒中病历文本识别
- 影像学报告OCR
- 手写脑卒中病历数字化
- 历史病历批量OCR处理

## 优势
- 端到端训练，无需字符级标注对齐
- 支持不定长文本识别
- CNN+RNN互补，特征提取和序列建模
- CTC解码灵活高效
- 工业部署成熟

## 局限性
- 对弯曲/透视变形文本效果差
- 不考虑字符间注意力关系
- 对严重遮挡/模糊文本效果有限
- CTC假设条件独立性
- 中文大字符集识别挑战

## 实现框架

# 路径优化（Path Optimization）

## 简介
路径优化是寻找最优路径的优化问题，在医疗领域用于救护车调度、医院内部物流、患者就诊路径规划等。

## 主要方法
- **精确算法**：Dijkstra、A*、动态规划
- **启发式算法**：遗传算法、蚁群算法、模拟退火
- **机器学习**：强化学习、深度学习

## 经典问题
- 最短路径问题
- 旅行商问题（TSP）
- 车辆路径问题（VRP）
- 多目标路径优化

## 医疗应用场景

### 院前急救调度
- 120派车最优路径
- 多救护车协同调度
- 动态路况路径规划
- 最近医院推荐

### 医院内部物流
- 药品配送路径
- 标本运输路径
- 医疗废物运输
- 设备调度路径

### 患者就诊规划
- 就诊科室顺序优化
- 检查路径规划
- 缩短等待时间

### 医疗资源配置
- 急救站点布局
- 医疗资源分配
- 区域医疗规划

## 优势
- 提高效率
- 节省时间
- 降低成本
- 优化资源配置

## 局限性
- 实时性要求高
- 动态变化处理困难
- 多目标权衡复杂
- 计算复杂度高

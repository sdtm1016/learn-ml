# 支持向量机（SVM）

## 简介
支持向量机（Support Vector Machine）是一种强大的监督学习算法，通过寻找最优超平面实现分类或回归。在医疗领域用于疾病诊断、生物标志物识别等。

## 核心原理
- **最大间隔**：寻找最大分类间隔的超平面
- **支持向量**：决定超平面的关键样本点
- **核函数**：将数据映射到高维空间
- **软间隔**：允许一定程度的误分类

## 常用核函数
- 线性核
- 多项式核
- RBF核（高斯核）
- Sigmoid核

## 医疗应用场景

### 疾病诊断
- 肿瘤分类
- 疾病鉴别诊断
- 病理分级
- 感染判断

### 生物标志物识别
- 疾病标志物筛选
- 基因表达分类
- 蛋白质组学分析

### 医学影像分析
- 影像分类
- 病灶识别
- 组织分类

### 检验数据分析
- 异常检测
- 分类诊断
- 风险分层

### 小样本学习
- 罕见病诊断
- 新疾病识别
- 小数据集分类

## 优势
- 在高维空间表现好
- 适合小样本学习
- 泛化能力强
- 理论基础扎实

## 局限性
- 对大规模数据效率低
- 核函数选择困难
- 参数调优复杂
- 对噪声敏感

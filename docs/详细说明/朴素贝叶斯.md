# 朴素贝叶斯（Naive Bayes）

## 简介
朴素贝叶斯是基于贝叶斯定理和特征条件独立假设的概率分类算法。因其计算简单、速度快，在医疗领域用于疾病诊断、文本分类等。

## 核心原理
- **贝叶斯定理**：P(A|B) = P(B|A)P(A)/P(B)
- **条件独立假设**：特征之间相互独立
- **最大后验概率**：选择概率最大的类别

## 主要类型
- **高斯朴素贝叶斯**：连续特征
- **多项式朴素贝叶斯**：离散特征
- **伯努利朴素贝叶斯**：二值特征

## 医疗应用场景

### 疾病诊断
- 疾病概率诊断
- 鉴别诊断
- 症状分析
- 风险评估

### 医学文本分类
- 病历分类
- 文献分类
- 意图识别
- 情感分析

### 检验结果分析
- 异常判断
- 诊断辅助
- 风险分层

### 垃圾信息过滤
- 医疗广告过滤
- 垃圾邮件识别
- 异常信息检测

### 快速筛查
- 疾病初筛
- 风险预警
- 分诊辅助

## 优势
- 计算速度快
- 需要训练数据少
- 处理多类别问题
- 对缺失数据不敏感

## 局限性
- 特征独立假设不现实
- 对输入数据表达形式敏感
- 分类边界不够灵活
- 零概率问题

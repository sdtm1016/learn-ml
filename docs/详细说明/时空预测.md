# 时空预测（Spatiotemporal Prediction）

## 简介
时空预测是同时考虑空间和时间维度的预测方法，在医疗领域用于疫情传播预测、急救资源热力预测、区域健康分析等。

## 核心概念
- **空间依赖**：相邻区域相互影响
- **时间依赖**：历史数据影响未来
- **时空交互**：空间和时间相互作用

## 主要方法
- **统计模型**：时空自回归、克里金
- **深度学习**：ST-GCN、ConvLSTM
- **图神经网络**：时空图卷积网络

## 常用模型
- ST-GCN：时空图卷积网络
- ConvLSTM：卷积LSTM
- DCRNN：扩散卷积RNN
- Graph WaveNet

## 医疗应用场景

### 疫情传播预测
- 流感传播预测
- 新冠疫情趋势
- 登革热流行预测
- 区域疫情预警

### 急救资源热力预测
- 120呼叫量预测
- 急救高峰预测
- 区域需求预测
- 资源调配预警

### 区域健康分析
- 疾病空间分布
- 健康风险地图
- 环境健康影响
- 区域差异分析

### 公共卫生决策
- 干预措施评估
- 资源配置优化
- 预警系统建设

## 优势
- 同时考虑时空因素
- 捕获复杂模式
- 提高预测准确性
- 支持区域决策

## 局限性
- 数据获取困难
- 模型复杂度高
- 计算资源需求大
- 外部因素影响大

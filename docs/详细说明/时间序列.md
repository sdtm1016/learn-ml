# 时间序列（Time Series）

## 简介
时间序列是按时间顺序排列的数据序列，在医疗领域广泛存在于生命体征监测、检验指标跟踪、疫情数据等场景。

## 核心特征
- **趋势**：长期变化方向
- **季节性**：周期性波动
- **周期性**：非固定周期波动
- **随机性**：不可预测的波动

## 分析方法
- **描述性分析**：统计特征、可视化
- **时序分解**：趋势、季节、残差分解
- **平稳性检验**：ADF检验、KPSS检验
- **自相关分析**：ACF、PACF

## 常用模型
- ARIMA：自回归积分滑动平均
- SARIMA：季节性ARIMA
- Prophet：Facebook开源工具
- LSTM：长短期记忆网络
- Transformer：注意力机制

## 医疗应用场景

### 生命体征监测
- 心率时序分析
- 血压变化趋势
- 血氧波动分析
- 体温监测

### 疫情预测
- 流感传播趋势
- 新冠疫情分析
- 登革热流行预测
- 高峰预警

### 检验指标跟踪
- 血糖变化趋势
- 肾功能监测
- 肝功能跟踪
- 肿瘤标志物监测

### 急诊拥挤度预测
- 急诊高峰预测
- 床位占用预测
- 就诊量预测

## 优势
- 反映数据变化趋势
- 支持预测分析
- 发现周期性规律
- 辅助决策制定

## 局限性
- 需要足够历史数据
- 对异常值敏感
- 长期预测准确性下降
- 外部因素影响大

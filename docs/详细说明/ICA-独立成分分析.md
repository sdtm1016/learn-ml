# ICA（独立成分分析）

## 简介
独立成分分析（Independent Component Analysis，ICA）是一种盲源分离技术，假设观测信号是多个独立源信号的线性混合，通过统计独立性准则分离出源信号。

## 核心原理
- **盲源分离**：从混合信号中分离独立源
- **独立性假设**：源信号相互独立
- **非高斯性**：利用非高斯性度量独立性

## 常用算法
- FastICA：快速不动点算法
- Infomax：信息最大化
- JADE：联合对角化
- Extended Infomax：扩展信息最大化

## 医疗应用场景

### fMRI脑功能网络分析
- 脑功能网络分离
- 默认模式网络识别
- 任务相关网络提取
- 静息态网络分析

### EEG信号处理
- 脑电伪迹去除
- 眼电、肌电分离
- 癫痫源定位
- 事件相关电位提取

### ECG信号处理
- 胎儿心电提取
- 心电噪声去除
- 心电信号分离

### 医学影像处理
- 图像去噪
- 特征提取
- 图像分离

## 优势
- 无需先验知识
- 分离独立信号源
- 适合盲源分离问题
- 计算效率较高

## 局限性
- 源数目估计困难
- 排列顺序不确定
- 幅度不确定
- 对噪声敏感

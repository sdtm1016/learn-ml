# LSTM（长短期记忆网络）

## 简介
LSTM（Long Short-Term Memory）是一种特殊的循环神经网络（RNN），由Hochreiter和Schmidhuber在1997年提出，专门用于解决长序列训练中的梯度消失问题。

## 核心结构
- **遗忘门（Forget Gate）**：决定丢弃哪些信息
- **输入门（Input Gate）**：决定存储哪些新信息
- **细胞状态（Cell State）**：长期记忆存储
- **输出门（Output Gate）**：决定输出哪些信息

## 主要变体
- Peephole LSTM
- GRU（门控循环单元）
- Bi-LSTM（双向LSTM）
- Stacked LSTM

## 医疗应用场景

### 时序预测
- ICU重症预警
- 疾病风险预测
- 生命体征监测
- 血糖预测

### 信号分析
- 心电图分析
- 脑电图分析
- 睡眠分期
- 癫痫检测

### 临床决策
- 脓毒症早期预警
- 呼吸衰竭预测
- 心跳骤停风险
- 休克预警

### 疫情预测
- 流感传播预测
- 新冠疫情趋势
- 登革热流行预测

## 优势
- 处理长序列能力强
- 捕获时序依赖关系
- 解决梯度消失问题
- 适合时间序列数据

## 局限性
- 计算效率较低
- 难以并行化
- 对超长序列效果有限
- 训练时间较长

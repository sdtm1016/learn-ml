# 强化学习（Reinforcement Learning）

## 简介
强化学习是一种通过与环境交互学习最优策略的机器学习方法，智能体通过试错获得最大化累积奖励。在医疗领域用于治疗方案优化、资源调度、临床决策支持等。

## 核心概念
- **智能体（Agent）**：学习决策的主体
- **环境（Environment）**：智能体交互的对象
- **状态（State）**：环境的状态描述
- **动作（Action）**：智能体的行为
- **奖励（Reward）**：行为的反馈信号

## 主要方法
- **基于价值**：Q-Learning、DQN
- **基于策略**：Policy Gradient、PPO
- **Actor-Critic**：结合价值和策略
- **Model-based**：学习环境模型

## 医疗应用场景

### 治疗方案优化
- 肿瘤化疗方案优化
- 靶向药选择
- 抗凝药剂量调整
- 抗生素治疗方案

### 医院资源调度
- 床位调度优化
- 急诊拥堵预测
- 手术室排程
- 医护人员排班

### 临床决策支持
- 机械通气参数调整
- 胰岛素剂量调节
- 血液透析方案

### 急救调度
- 120派车优化
- 救护车路径规划
- 急救资源分配

## 优势
- 学习最优策略
- 适应动态环境
- 无需标注数据
- 考虑长期收益

## 局限性
- 样本效率低
- 训练不稳定
- 医疗场景风险高
- 难以直接在患者身上试验
